What's CypherPunk Radio?

CypherPunk Radio is an indipendent internet radio focused on being private, secure & unstoppable.

What will we hear?

Music of a lot of different genres, live podcasts and Monero talking :)

How to listen to us?

CLI| GUI| Android

MPlayer

Tor

- Installtorsocks
- Run: torsocks mplayer -prefer-ipv4 http://channel.tormirror

I2P

- Installi2pd
- Run: export http_proxy="http://127.0.0.1:4444"
- Run: mplayer -prefer-ipv4 http://i2pmirror/channel

Issues

- Not consistent stream?
  Put -cache-min 99 after mplayer

VLC

Tor

- Install Tor
  - CTRL + P on the main screen
   Show setting in the left corner to All
   > Input / Codecs
   scroll till you find SOCKS server input 127.0.0.1:9050
  - CTRL + N on the main screen and input http://channel.tormirror.

I2P

- Install i2pd
  - CTRL + P on the main screen
> Input / Codecs
  scroll till you find HTTP proxy input http://127.0.0.1:4444
  - CTRL + N on the main screen and input http://i2pmirror/channel.
